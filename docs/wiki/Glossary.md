# Glossary

| Term | Definition |
|------|-----------|
| **btrfs** | Copy-on-write filesystem. Enables instant snapshots and rollbacks. |
| **Caddy** | Modern reverse proxy with automatic HTTPS and simple config syntax. |
| **ed25519** | Modern SSH key type. Faster and more secure than RSA. |
| **Gaia** | AMD's open-source AI agent framework for Ryzen AI hardware. |
| **GGUF** | File format for quantized LLM models used by llama.cpp. |
| **gfx1151** | AMD GPU architecture identifier for Strix Halo's Radeon 8060S. |
| **HIP** | AMD's GPU programming interface, similar to CUDA. |
| **Lego Block** | A modular, independent service that snaps in and out of the stack. |
| **Lemonade SDK** | AMD's unified AI backend wrapping llama.cpp, whisper, kokoro, and more. |
| **llama.cpp** | High-performance LLM inference engine in C/C++. |
| **LLM** | Large Language Model — the AI that generates text. |
| **Mesh** | Network of machines that can all SSH to each other. |
| **NPU** | Neural Processing Unit — dedicated AI accelerator on Ryzen AI chips. |
| **pyenv** | Python version manager. Lets you install Python 3.13 alongside system Python. |
| **ROCm** | AMD's GPU compute platform, equivalent to NVIDIA's CUDA. |
| **Strix Halo** | AMD's high-end Ryzen AI MAX+ processor with 128GB unified memory. |
| **systemd** | Linux service manager. Starts, stops, restarts, and monitors services. |
| **VRAM** | Video RAM — memory available to the GPU for model loading. |
| **venv** | Python virtual environment. Isolates packages from the system. |
| **Vulkan** | Cross-platform GPU API. Alternative to ROCm for inference. |
| **XDNA** | AMD's NPU architecture and driver. |
